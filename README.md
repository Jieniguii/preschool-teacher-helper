# 🦞 幼师助手 — 观察记录 & 期末报告生成器

> 填口语 → 出专业记录。让幼儿园老师的文案工作不再耗时。

## 这是什么？

一个专为幼儿园老师设计的桌面工具，解决两个高频痛点：

| 功能 | 原来 | 现在 |
|------|------|------|
| **观察记录** | 手写流水账，缺专业分析 | 用口语随便记 → 点生成 → 出四段式专业记录 |
| **期末报告** | 逐个孩子写几千字，耗时巨大 | 汇总观察片段 → 一键生成六段式完整报告 |

**智能模板引擎**：能识别你描述里的关键词（坚持、分享、情绪、创造……），自动匹配最合适的分析话术。每个场景配了多套表述，每次生成都不重样。参照《3-6 岁儿童学习与发展指南》，3-4/4-5/5-6 岁各有独立的发展指标库。

## 怎么使用？

### 前提：安装 Python 3

电脑需要有 Python 3。[下载地址](https://www.python.org/downloads/) → 下载安装包 → **安装时务必勾选底部的「Add Python to PATH」** → 装完就行。

> 只装一次，以后不用管。

### 启动（2 步）

1. 下载本仓库 ZIP → 解压到电脑任意位置
2. 双击 `启动.bat` → 浏览器自动打开，开始用

> **不要**直接双击 `index.html`，一定要通过 `启动.bat` 打开！

### 数据存储

所有记录保存在 `data/records/` 文件夹。每个孩子一条 `.json` 文件，可以随时查看、复制、备份。

> ⚠️ 每学期末建议把 `data` 文件夹复制到 U 盘或微信发给自己，防止电脑出问题丢数据。

## 功能说明

### 📝 观察记录

**填 5 项 → 点生成 → 完工：**

1. 孩子姓名
2. 年龄（小班/中班/大班）
3. 观察场景（区角/户外/集体教学/自由游戏/生活环节/同伴交往/艺术创作）
4. **发生了什么** — 用口语随便写，真实最重要。例："搭积木桥倒了 3 次，每次都重新来，第 4 次成功，兴奋地叫小红来看"
5. 语言/行为细节（选填）

**生成四段式记录**：客观描述 → 行为分析 → 发展解读 → 支持建议

生成后可一键复制、下载 Word、或保存到历史（期末汇总用）。

### 📊 期末报告

1. 切到「历史记录」Tab → 找到对应孩子 → 点「导入报告」
2. 勾选重点关注领域、填写家长关注点
3. 点生成 → 六段式完整报告（概述 → 五大领域分析 → 亮点 → 建议 → 下学期重点 → 致家长寄语）

### 📂 历史记录

每次观察记录点「保存到历史」后，数据存在 `data/records/` 文件夹，可查看、导入报告、删除。

## 技术说明

- 纯 Python 3 标准库，**零额外依赖**，不需要 `pip install` 任何东西
- 前端 HTML/CSS/JS，适配电脑和手机
- 数据 JSON 文件存储在本地，可备份可迁移
- 不联网，不注册，不留痕
