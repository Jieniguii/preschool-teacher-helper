# 🦞 幼师助手 — 观察记录 & 期末报告生成器

> 让专业记录不再耗时，幼儿园老师的好帮手。

## 这是什么？

一个专为幼儿园老师设计的网页工具，解决两个高频痛点：

| 功能 | 原来 | 现在 |
|------|------|------|
| **观察记录** | 手写流水账，缺专业分析 | 填口语 → 点生成 → 出四段式专业记录 |
| **期末报告** | 逐个孩子写几千字，耗时巨大 | 汇总观察片段 → 一键生成完整报告 |

所有内容参照《3-6 岁儿童学习与发展指南》，专业、真实、不模板化。

## 怎么使用？

### 方式一：在线使用（推荐）

直接浏览器打开：**https://jieniguii.github.io/preschool-teacher-helper/**

不用安装，不用注册，手机电脑都能用。

### 方式二：下载到本地

点击仓库右上角 **Code → Download ZIP**，解压后双击 `index.html` 即可。

## 功能详解

### 📝 观察记录

**第 1 步：填写信息**

- **孩子姓名** — 填名字就行
- **年龄** — 选 小班 / 中班 / 大班
- **观察日期** — 自动填今天，可改
- **观察场景** — 点选一个（区角活动、户外活动、集体教学、自由游戏、生活环节、同伴交往、艺术创作）
- **发生了什么** — 用口语随便记几句，不用修饰，真实最重要
- **语言/行为细节** — 有印象就写，没有就空着

**第 2 步：点击「生成观察记录」**

系统自动产出四个部分：

1. **客观描述** — 保留你的原始观察，去掉主观评价词
2. **行为分析** — 根据场景匹配专业分析框架
3. **发展解读** — 引用《指南》中对应年龄段的发展指标
4. **支持建议** — 给出可操作的家园配合方案

**第 3 步：使用结果**

- 📋 **一键复制** — 粘贴到幼儿园的 Word 模板里
- 💾 **保存到历史** — 存在本地，期末汇总用
- 📥 **下载 Word** — 导出为 .doc 文件

### 📊 期末报告

**第 1 步：收集素材**

两种方式：
- **从历史导入** — 切到「历史记录」Tab，找到对应孩子的记录，点「导入报告」
- **手动输入** — 直接粘贴本学期的观察片段，每条一段

**第 2 步：填报告信息**

- 孩子姓名、年龄、学期
- **重点关注领域** — 勾选需要重点分析的领域（健康/语言/社会/科学/艺术）
- **家长关注点** — 比如"吃饭挑食""不爱说话"，生成时会针对性回应

**第 3 步：点击「生成期末评估报告」**

系统产出六段式完整报告：

1. 本学期整体发展概述
2. 五大领域发展分析
3. 本学期成长亮点
4. 待发展方面与教育建议
5. 下学期重点关注
6. 致家长的温馨寄语

### 📂 历史记录

每次观察记录点击「保存到历史」后，会自动存储在这里。

- 查看：点击「查看」弹窗查看完整记录
- 导入报告：期末做报告时一键导入
- 删除：不需要的记录可以删掉

> ⚠️ 数据存在浏览器本地，换电脑或清缓存会丢失。重要的期末报告建议下载保存。

## 使用小贴士

### 观察记录怎么写效果好？

- **关键在"发生了什么"** — 写得越具体，生成的记录越真实
- 例（好）："搭积木桥倒了 3 次，每次都重新来，第 4 次成功，兴奋地叫小红来看"
- 例（差）："搭积木表现很好"
- **语言/行为细节是加分项** — 写了孩子说的话或做的动作，报告会更生动

### 期末报告怎么最省时？

平时积累 + 期末汇总 = 最高效。建议每次做完观察记录后顺手「保存到历史」，期末直接导入。

### 这份记录能直接交给家长吗？

能。生成的内容参照《3-6 岁儿童学习与发展指南》，用语专业、温暖、不套话。建议在生成后根据自己的实际印象微调，让它更贴合你眼中的那个孩子。

## 技术说明

- 纯前端，无服务器，数据只存在你的浏览器里
- 适配手机和电脑，户外观察用手机也能操作
- 生成逻辑：专业模板引擎 + 场景适配框架，非 AI 联网生成，内容稳定可控
